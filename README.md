# ⏱️ React Time Dashboard

A modern **Time Dashboard** built using **React.js and Tailwind CSS** that includes:

* 🇮🇳 **Live Indian Standard Time Clock**
* ⏳ **Countdown Timer with Quick Timers**
* ⏱️ **Stopwatch with Millisecond Precision**
* 🎨 **Modern Dashboard UI**
* 📱 **Responsive Layout**

This project demonstrates **React Hooks (`useState`, `useEffect`)** and **Tailwind CSS UI design**.

---

# 🚀 Features

### 🕒 Live Clock

Displays **real-time Indian Standard Time (IST)** updated every second.

### ⏳ Countdown Timer

* Custom seconds input
* Start / Stop / Reset controls
* Alert notification when timer finishes
* Quick timer buttons:

  * 5s
  * 10s
  * 30s
  * 60s
  * 120s

### ⏱️ Stopwatch

* Millisecond precision stopwatch
* Start / Stop / Reset functionality
* Time formatted as:

```
MM : SS : MS
```

### 🎨 Modern UI

* Built using **Tailwind CSS**
* Professional dark dashboard theme
* Responsive layout
* Card-based UI design
